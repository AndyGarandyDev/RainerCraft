# Rainer Craft

## Features

    Rainers Weisheiten und Kampfschreie schallen nun durch die Welt von Minecraft. 
    Für die größten Haider unter euch gibt es sogar einen Mini Rainer für unterwegs.

- **Textures**
    - Totem
        - Rainer Skin
- **Sounds**
    - Mobs
        - Zombie
        - Skelett
        - Zombie Piglin
        - Creeper
    - Spieler
        - Schaden austeilen
            - versch. Sounds für schwache, starke, kritische und swipe Treffer
        - Schaden erhalten
            - normale Treffer
            - Berrybush
            - Fire/Freeze
            - Ertrinken
        - Items
            - Glocke ( Rainer verteidigt sein Dorf und Grundstück )
            - Goat Horn ( Rainer Frenzy )

## Anleitung
- **Minecraft Mod Ordner öffnen** 
  - Esc drücken ...
  - ... dann auf *Optionen* ... 
  - ... dann auf *Ressourcenpakete* ... 
  - ... und auf *Packetordner öffnen* klicken
- **Mod importieren**
  - Die Datei *Rainers Paarungsrufe.zip* in den Minecraft Mod Ordner kopieren
- **Mod aktivieren**
  - Zurück ins Spiel ...
  - ... nun erscheint Rainer in Fußballmontur unter den verfügbaren Mod ... 
  - ... auf das Bild klicken ... 
  - ... und ***Meddl, Loide!***
